# WeedsFinder

Cannabis strain database, law guides, and dosing tools.

**Live site:** [weedsfinder.com](https://weedsfinder.com)

## Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- Deployed on [Vercel](https://vercel.com/giotskr-1862s-projects/weedsfinder)

## Develop locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` deploy to the **weedsfinder** Vercel project (`giotskr-1862s-projects` team).

GitHub Actions needs a `VERCEL_TOKEN` secret from the Vercel account that owns the weedsfinder project:
[vercel.com/account/tokens](https://vercel.com/account/tokens)
